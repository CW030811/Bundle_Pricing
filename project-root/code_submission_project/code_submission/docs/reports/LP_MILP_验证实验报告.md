# LP-MILP Improvement 验证实验报告

## 一、实验概述

**目的**：验证 Local Search 过程中每次 LP-guided improvement 是否真实有效（即是否在 MILP Solver 下同样有提升）。

**规模**：6 个数据集 × 10 个样本 = 60 个样本

**配置**：
- 随机种子：np=42, torch=42
- max_iterations=50, tolerance=1e-6
- 策略：Global Top-K (K=ceil(sqrt(m)))

---

## 二、核心结论

| 指标 | 数值 |
|------|------|
| **总体转化成功率** | **97.54%** |
| 总改进次数 | 610 |
| 成功转化为 MILP 提升 | 595 |
| 失败次数 | 15 |

**结论**：LP-guided improvement 在绝大多数情况下（97.54%）能成功转化为 MILP 提升，验证了 LP 作为快速邻域评估的有效性。

| 步骤 | 总次数 | 成功次数 | 成功率 | 备注 |

| 1 | 58 | 58 | 100% | 第一步改进全部成功 |

**规律**：前 2 步 100% 成功；第 3、5、10 步成功率较低；后续步骤多数为 100%。可能原因：早期改进的 bundle 集合变化较小，LP 与 MILP 一致性好；中期某些 neighbor 在 MILP 下可重分配导致单步未提升。

## 八、总结与建议

1. **LP 有效性**：总体 97.54% 的转化成功率表明 LP 作为 Local Search 的快速评估手段是可靠的。
2. **失败原因**：MILP 可在同一 bundle 集合内重分配 segment，导致某些 LP 认为的改进在 MILP 下未体现；长搜索路径下该效应更明显。
3. **建议**：当前 LP-guided Local Search 策略可继续使用；若需更高可靠性，可在关键步骤增加 MILP 校验（会带来额外计算成本）。
