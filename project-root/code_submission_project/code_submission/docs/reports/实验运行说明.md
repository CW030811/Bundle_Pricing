## 实验流程

1. **初始预测**: 使用训练好的 Edge-Scoring GCN 模型生成初始 bundle 预测
2. **初始 MILP 求解**: 对初始预测进行 MILP 优化，获得基准收入
3. **Local Search**: 使用混合 MILP-LP 策略进行局部搜索优化
   - 生成邻域分配（Add/Drop 操作）
   - 使用 LP 快速评估邻域
   - 贪心策略接受改进
4. **最终 MILP 验证**: 对优化后的结果进行最终 MILP 验证

## 结果指标

每个 CSV 文件包含以下列：
- `n_products`: 产品数量
- `revenue_ratio`: 收入比率（相对于最优解）
- `runtime_ratio`: 运行时间比率（相对于基准方法）
- `total_time`: 总运行时间
- `base_running_time`: 基准运行时间
- `threshold_time`: GCN 推理时间
- `initial_milp_time`: 初始 MILP 求解时间
- `local_search_time`: Local Search 时间
- `initial_revenue`: 初始收入比率
- `improvement`: 改进幅度
- `iterations`: 迭代次数
- `improvements`: 改进次数
- `lp_solver_calls`: LP 求解器调用次数
- `milp_solver_calls`: MILP 求解器调用次数
