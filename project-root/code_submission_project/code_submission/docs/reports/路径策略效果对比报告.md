# Local Search 路径策略效果对比报告

## 实验概述

**实验日期**: 2024年  
**测试策略**: 
1. **原策略 (Segment-based)**: 每轮生成 2*m 个邻域
2. **新策略 (Global Top-K)**: 每轮生成最多 2*K 个邻域，K = ceil(2 * sqrt(m))

**数据集**:
- m10_n10 (m=10, n=10) - 两种策略都有测试
- BSP_m10n15 (m=10, n=15) - 两种策略都有测试
- BSP_m10n20 (m=10, n=20) - 两种策略都有测试
- BSP_m10n25 (m=10, n=25) - 两种策略都有测试
- BSP_m15n15 (m=15, n=15) - 两种策略都有测试
- m20_n10 (m=20, n=10) - 两种策略都有测试
- BSP_m20n15 (m=20, n=15) - 两种策略都有测试
- m30_n10 (m=30, n=10) - 仅Global Top-K
